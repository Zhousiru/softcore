一个 mc fabric 服务器 only mod，作用是提供一个类 hardcore 模式的玩家体验。

具有以下功能：

- 将当前模式设置为 hardcore 模式，显示 hardcore 模式特殊血量图标和复活界面
- 玩家死亡重生后，设置为旁观者模式，并且禁止玩家移动（称为封禁）
  - 此时 actionbar 显示“你已死亡，将在 XX 天 XX 小时 XX 分 XX 秒后复活”（每秒更新）
  - 复活时间可以在配置文件中定义，支持“动态时限”
    ```toml
    [respwan]
    cool_down_rules = [
      { cond = "1 in 48h", time = "0" }, # 48 小时窗口内第 1 次死亡，不封禁，直接重生（从上到下，这条优先级最高）
      { cond = "1 in 24h", time = "2h" }, # 24 小时窗口内第 1 次死亡，封禁 2 小时
      { cond = "2 in 24h", time = "4h" }, # 24 小时窗口内第 2 次死亡，封禁 4 小时
      { cond = "else", time = "1h" }, # 其余，封禁 12 小时
    ]
    ```
- 如果玩家死亡重生后，在封禁模式下，并且有其他玩家在线
  - 此时 actionbar 显示“你已死亡，将在 XX 天 XX 小时 XX 分 XX 秒后复活，按 Shift 切换到其他玩家视角”（每秒更新）
  - 侦测玩家按下 shift（考虑这里的实现方法，向下移动、脱离当前旁观视角），可以在出生点、在线玩家 A、在线玩家 B、……之间循环切换
  - 切换到玩家视角后，如果被旁观玩家死亡、退出，循环切换
  - 如果当前切换到了玩家视角，此时 actionbar 显示“你已死亡，正在旁观 XXX，将在 XX 天 XX 小时 XX 分 XX 秒后复活，按 Shift 切换到其他玩家视角”
  - 考虑是否还有其他边界情况
